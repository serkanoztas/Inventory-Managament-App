# Inventory Management App

A full-stack Inventory Management Application built with React, Node.js, Express.js, and MongoDB. The application provides inventory tracking, user authentication, role-based access control, and a dedicated admin dashboard for managing products, categories, users, and carriers.

## Features

* JWT Authentication & Authorization
* Secure password hashing with Bcrypt
* Role-based access control
* Dedicated Admin Dashboard
* Product Management (CRUD)
* Category Management (CRUD)
* User Management (CRUD)
* Carrier Management (CRUD)
* Inventory Tracking
* RESTful API Architecture
* Responsive User Interface

## Installation

### Clone the repository

```bash
git clone https://github.com/serkanoztas/Inventory-Managament-App.git
```

### Navigate to the project directory

```bash
cd Inventory-Managament-App
```

### Install dependencies

Frontend:

```bash
npm install
```

Backend:

```bash
npm install
```

### Configure environment variables

Create a `.env` file inside the backend directory:

```env
PORT=3000
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
```

### Run the application

Backend:

```bash
npm run dev
```

Frontend:

```bash
npm run dev
```
